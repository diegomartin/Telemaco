Bienvenido al servidor del proyecto Telemaco

Información de funcionamiento del servidor.

  • Desde el menú de la izquierda se puede acceder a la interfaz de administración del servidor, navegar en la información disponible en la base de datos o revisar la documentación generada por Django-Admin.
  • La aplicación móvil debe incluir en el parámetro de configuración server_url del fichero Telemaco/res/values/strings.xml la URL de conexión al servidor http://localhost:8000/telemaco_api/
  • El proceso data_fetch.py debe ser ejecutado periódicamente para que la base de datos se mantenga actualizada.
  • Puede usarse manage.py para realizar otras tareas de mantenimiento.
  • Para más información, consulte el manual del usuario.
